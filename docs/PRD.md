# PRD (Product Requirement Document)

이 문서는 **생기부 개인정보 익명화 도구**의 제품 요구사항 및 범위를 정의합니다.

---

## 1. 프로젝트 개요

* **프로젝트명**
  * 생기부 개인정보 익명화 도구
* **프로젝트 목적**
  * Excel 기반 학교생활기록부 파일에서 학생 이름 및 학교명을 자동 탐지하고 익명화하여 개인정보 보호 및 외부 제출 작업을 효율화합니다.
* **핵심 문제**
  * 생기부 문서 내 개인정보를 수작업으로 수정하는 과정이 반복적이고 많은 시간 소요.
  * 수작업 수정 시 개인정보 유출(누락) 및 오타 위험 존재.
  * 완전 자동 삭제 방식은 오탐 위험이 있어 별도의 사용자 검수 과정이 반드시 필요.
* **타겟 사용자**
  * 중학교 및 고등학교 생기부 업무 담당 교사
  * 생활기록부 검수 및 외부 제출 업무 관련 행정 사용자
* **핵심 가치 제안**
  * 개인정보 익명화 자동화 지원
  * 검수 기반의 안전한 승인/수정 처리
  * 원본 데이터 보존 중심의 안전한 파일 처리 구조
  * 별도 설치 없이 실행 가능한 Windows 로컬 단독 실행형 업무 툴

---

## 2. 요구사항 및 MVP 범위

### 핵심 기능 (In Scope)
* **Excel (.xlsx) 파일 로드**: 단일 및 다중 파일 일괄 추가/선택 처리.
* **익명화 규칙 정의**:
  * **학생명 가명화**: 지정된 학생 이름 매핑 처리 (예: 김민수 ➡️ 학생1, 이서연 ➡️ 학생2).
  * **학교명 가명화**: 지정된 학교명 매핑 처리 (예: 서울중학교 ➡️ 학교A).
  * **단어 완전 삭제**: 특정 단어 검색 및 완전 제거/대체 텍스트 치환 처리.
* **결과 미리보기 및 검수**: 탐지 결과를 테이블 형태로 확인하며, 사용자가 개별 항목을 '승인'하거나 '치환 내용 수동 편집' 지원.
* **Safe Save 출력**: 원본 파일을 안전하게 보존하며, 검증 단계를 거쳐 `_anonymized.xlsx` 접미사가 붙은 새 파일로 저장.
* **매핑 대장 저장**: 익명화 처리 전후의 가명화 매핑 정보를 선택적으로 CSV/Excel 포맷으로 저장.

### 제외 범위 (Out of Scope)
* OCR 기능 (이미지 문서 판독)
* PDF 및 한글(HWP) 처리 (향후 확장 검토)
* AI/머신러닝 기반 자동 개인정보 추론 (100% 예측 가능한 규칙 기반 매칭 우선)
* 클라우드 동기화 및 사용자 계정 시스템 (안전한 오프라인 단독 로컬 동작 보장)

## 3. 사용자 및 UX 흐름

### 주요 시나리오
1. 사용자가 프로그램을 실행합니다.
2. 익명화 처리를 실행할 대상 Excel 파일들을 추가합니다 (다중 선택 및 순서 보존 지원).
3. 학생 이름 목록, 학교명 목록, 삭제할 단어 목록을 텍스트 입력 칸에 줄바꿈 또는 쉼표(,)로 구분하여 입력합니다.
4. "개인정보 자동 탐지 실행"을 클릭하면 비동기 스레드에서 Excel 분석을 수행합니다.
5. 하단 검수 테이블에 매칭된 셀 주소, 원본 값, 변경 예정 값, 적용 여부(체크박스) 목록이 로드됩니다.
6. 사용자는 틀린 항목을 교정하거나 적용 여부를 체크 해제합니다.
7. "익명화 결과 파일 최종 저장" 버튼을 클릭하여 결과물이 저장될 폴더를 지정한 후 Safe Save 파이프라인으로 안전하게 저장합니다.
8. 필요한 경우 매핑 정보 저장 옵션을 체크하여 매핑 대장을 함께 출력합니다.

### 검수 테이블 사양
* 컬럼 구조: `시트명` | `셀 주소` | `원본 값` | `변경 예정` | `적용 여부 (체크박스)`
* 사용자는 테이블의 `변경 예정` 열을 직접 더블 클릭하여 수동 수정이 가능해야 합니다.

## 4. 비기능 요구사항
* **안정성 (최우선)**: 저장 과정에서 Excel 수식, 스타일 서식 등이 깨지거나 원본 데이터가 소실되는 현상이 전혀 없어야 함.
* **성능**: 대용량 Excel 파일 처리 시 UI 스레드가 정지(Freeze)되지 않도록 백그라운드 Worker 스레드 기반으로 연산 처리.
* **호환성**: Windows 10/11 환경에서 추가 라이브러리 설치 없이 실행할 수 있는 독립형 `.exe` 패키지 제공.
