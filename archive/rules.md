# 🚀 생기부 개인정보 익명화 프로젝트용 AI 협업 지침 v3.0

⚠️ 본 지침은 Windows 기반 Python 데스크톱 프로그램(Excel 개인정보 익명화 도구)을 AI와 안정적으로 협업 개발하기 위한 실전 운영 규칙이다.

프로젝트 특성상:
- 개인정보 처리
- Excel 파일 무결성
- 검수 기반 워크플로우
- exe 배포
- 학교 업무 환경

을 핵심 전제로 하며,
속도보다 데이터 안정성과 복구 가능성을 우선한다.

---

# 0. 프로젝트 핵심 목표

## 프로젝트 목적
Excel(.xlsx) 기반 생기부 파일에서:
- 학생 이름
- 학교명

을 자동 탐지하고,
검수 후 익명화 처리하는 Windows 데스크톱 프로그램 개발.

- PyInstaller

3. 검수 가능한 UI 우선
